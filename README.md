## 项目简介

用户留存率是衡量产品用户质量和用户黏性的核心指标之一。
本项目基于 Cohort Analysis(同期群分析)方法，
对用户在首次活跃后的第 N 天留存情况进行分析，
并通过构建留存矩阵和热力图，直观展示不同用户群体的留存表现。

---

## 项目目标

- 计算用户第 N 天的留存率；

- 构建留存率矩阵 (Retention Matrix);

- 利用热力图展示用户留存变化趋势；

- 根据留存结果分析用户行为特征，并提出业务优化建议。

## 分析流程

- 准备数据：

计算每位用户的首次注册日，将其合并到原表中
接着计算出用户的留存天数，并返回

- 计算任意一天的留存率

留存率 = 某天仍活跃的用户人数 / 新增人数
首先先计算出新增人口数
接着计算出用户留存人数，
根据公式算出留存率，并将其封装成函数进行复用

- 计算留存率矩阵

首先统计所有天数的用户留存人数，
接着将其展开成列，
最后，利用每行的留存值与每行第一个值，即cohort相除得到留存率

- 可视化

将留存率保存到csv文件中，并绘制热力图显示结果。

---

## 核心结果

- 2025-1-1日注册日，用户留存情况由100%逐渐降至第一天、第二天的50%，到第七天降至0；

- 2025-1-2日注册日，用户留存情况在第一天，第七天都能够保持100%，用户留存表现较好；

- 2025-1-5日注册日，用户留存情况首日之后，降至为0，用户留存表现较差。
